# Tomb of Nassak An-Rah

A [Vaults of Vaarn](https://vaultsofvaarn.com/) text adventure with old-school
terminal vibes, playable in the browser: the Python game engine runs
client-side in WebAssembly (Pyodide). No server, no account -- saves live in
your browser.

**Play it: https://ccb.github.io/tomb-of-nassak-an-rah/**

Built with the `text_adventure_games` engine (CIS 7000 summer research,
University of Pennsylvania). This repo holds only the built site; it is
deployed from the (private) development repo.
